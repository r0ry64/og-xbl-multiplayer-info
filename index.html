OG Xbox Live Multiplayer Game Info/Features

(Insignia supported games only)

**Work in progress** Last update: 2026-01-23

Legend

Pre-game lobby Supports a pre-game "waiting room" lobby before the match begins.
Join in progress Supports joining a multiplayer lobby while the match is in progress, either as a spectator or player.
Max Players Max online players allowed in a lobby according to game mode. Some games have increased player limit with a dedicated server.
Communicator Chat Type Global: Talk and listen to everyone in the match. Only applies when a match in session, pre-game lobbies are always global. Some games put dead players and spectators in a seperate voice channel.
Team: Talk and listen to your teammates only.
Push-to-talk: Push and hold a specific button to talk.
Proximity: Talk and listen to players near you only.
NAT 1 Recommended: If you don't have Open NAT type (NAT 1), you might have issues joining or might block others from joining. PORT FORWARD 3074 UDP TO GET NAT 1! Open NAT type is highly recommended for all games, but some games are more strict with NAT than others.
NAT 1 / NT:1 = Open NAT Type
NAT 2 / NT:2 = Moderate NAT Type
NAT 3 / NT:3 = Strict NAT Type
Required: You MUST have Open NAT type (NAT 1) or else you will have issues joining, or you will block others from joining.
DLC Free: Downloadable content was free and is provided by Insignia. Some games have both free and paid DLC, Insignia will only be providing the free DLC.
Paid: Downloadable content was paid, and will not be provided by Insignia.
Netcode: Ok: May experience lag/delay, rubberbanding/jittery gameplay, collision glitches. Mostly noticable on higher pings. Still very playable.
Bad: Very noticable lag, slow input delay, frame delay, clearly unfair host advantage. May be unplayable on higher pings.
